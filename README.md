# Скальпинг-бот Binance Futures (Paper Trading)

## Что это

Бот для автоматического скальпинга альткоинов на Binance Futures.
Работает в режиме **Paper Trading** — торгует на реальных ценах,
но без реальных денег. Записывает каждую сделку.

## Быстрый старт

```bash
cd scalping_bot
pip install -r requirements.txt
python main.py          # запуск
                        # Ctrl+C — остановка + итоговый отчёт
```

## Как бот принимает решение

### 1. Автоматический отбор монет
Сканирует все ~600 фьючерсных пар на Binance.
Исключает фундаментальные (BTC, ETH, SOL...) и стейблкоины.
Отбирает топ-7 альткоинов по объёму, спреду, волатильности.
Пересканирует каждый час.

### 2. Мультитаймфрейм-анализ (MTF)
Для каждой монеты из watchlist:

- **1h** — находит ключевые уровни (поддержка/сопротивление)
- **15m** — проверяет тренд (совпадает ли с направлением сделки?)
- **5m** — ищет сигнал (bounce или breakout у MTF-уровня)
- **1m** — уточняет вход (ждёт реакцию для точного входа)

Уровни с разных ТФ объединяются. Уровень видимый на 1h И 15m —
самый сильный (силы складываются с весами x3 и x2).

### 3. Фильтры
- Risk/Reward минимум 1:2
- Объём подтверждает движение
- BTC-корреляция (если BTC падает — лонги на альтах блокируются)
- Только strong и medium сигналы (weak отсеиваются)

### 4. Управление позицией
- Размер от риска: 1% депозита на сделку
- Стоп-лосс и тейк-профит автоматически
- Перенос стопа в безубыток при +0.5%

## Логи

Всё пишется в папку `logs/`:

| Файл | Содержимое |
|------|-----------|
| `bot_YYYY-MM-DD.log` | Полный лог работы бота (все события) |
| `trades_YYYY-MM-DD.csv` | Таблица сделок (для Excel) |
| `trades_YYYY-MM-DD.json` | Сделки в JSON (полные данные) |
| `signals_YYYY-MM-DD.json` | Все обнаруженные сигналы |

## Настройка

Все параметры в `config.py`:

- `SCANNER_TOP_N` — сколько монет отбирать (7)
- `INITIAL_BALANCE` — стартовый баланс для симуляции (1000 USDT)
- `RISK_PER_TRADE` — риск на сделку (0.01 = 1%)
- `LEVERAGE` — плечо (10)
- `EXCLUDED_FUNDAMENTAL` — какие монеты исключить

## Структура файлов

```
config.py          — Настройки
log_setup.py       — Централизованное логирование
exchange.py        — Подключение к Binance (ccxt.pro)
coin_scanner.py    — Автоматический отбор монет
levels.py          — MTF-уровни (1h + 15m + 5m)
volume_analyzer.py — Анализ объёма
signals.py         — MTF-сигналы (15m тренд + 5m паттерн + 1m вход)
trader.py          — Paper Trading (симуляция)
logger_mod.py      — Запись сделок в JSON + CSV
analytics.py       — Отчёты и аналитика
main.py            — Точка входа
```
