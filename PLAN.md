# Скальпинг-бот Binance Futures — План проекта

## Режим: Paper Trading (без реальных денег)

Бот работает на РЕАЛЬНЫХ данных с биржи, но сделки только симулирует.
Записывает каждый вход/выход, считает P&L.

---

## Архитектура (8 модулей)

```
scalping_bot/
├── config.py          # Настройки бота (монеты, таймфреймы, риски)
├── exchange.py        # Подключение к Binance через ccxt.pro
├── levels.py          # Определение уровней поддержки/сопротивления
├── volume_analyzer.py # Анализ объёма (всплески, дельта, подтверждение)
├── signals.py         # Генерация сигналов (отскок / пробой)
├── trader.py          # Paper Trading — симуляция сделок
├── logger_mod.py      # Логирование сделок в файл
├── analytics.py       # Аналитика: winrate, P&L, лучшие сетапы
├── main.py            # Точка входа — запуск бота
└── requirements.txt   # Зависимости
```

---

## Шаги реализации

### Шаг 1: Инфраструктура
- [x] config.py — все настройки в одном месте
- [x] exchange.py — подключение к Binance, получение свечей и сделок
- [x] requirements.txt

### Шаг 2: Аналитическое ядро
- [x] levels.py — поиск уровней (адаптивное окно + ATR)
- [x] volume_analyzer.py — всплески, подтверждение тренда, ложные пробои

### Шаг 3: Торговые сигналы
- [x] signals.py — bounce + breakout детекторы с фильтрами

### Шаг 4: Исполнение и логирование
- [x] trader.py — paper trading, расчёт позиции, стоп/тейк
- [x] logger_mod.py — запись каждой сделки в JSON + CSV

### Шаг 5: Аналитика и запуск
- [x] analytics.py — итоги дня
- [x] main.py — основной цикл бота
