# Скальпинг-бот Binance Futures (Paper Trading)

## Что это?

Бот для автоматического скальпинга на Binance Futures.
Работает в режиме **Paper Trading** — торгует на реальных ценах,
но **без реальных денег**. Записывает каждую сделку.

## Быстрый старт

### 1. Убедись что установлен Python 3.12+

```bash
python --version
```

### 2. Установи зависимости

```bash
cd scalping_bot
pip install -r requirements.txt
```

### 3. Настрой бота (по желанию)

Открой `config.py` и поменяй:
- `WATCHLIST` — какие монеты торговать
- `INITIAL_BALANCE` — стартовый баланс для симуляции
- `RISK_PER_TRADE` — риск на сделку (0.01 = 1%)
- `LEVERAGE` — плечо

### 4. Запусти

```bash
python main.py
```

### 5. Останови

Нажми `Ctrl+C` — бот покажет итоговый отчёт.

## Что бот делает

Каждые 5 секунд для каждой монеты из watchlist:

1. **Загружает свечи** с Binance (реальные данные)
2. **Находит уровни** поддержки и сопротивления
3. **Анализирует объём** — всплески, подтверждение, ложные пробои
4. **Ищет сигналы**:
   - Отскок от уровня (bounce)
   - Пробой уровня (breakout)
5. **Фильтрует** через BTC-корреляцию и Risk/Reward
6. **Симулирует сделку** — записывает вход, стоп, тейк
7. **Следит** — сработал ли стоп или тейк
8. **Логирует** всё в JSON и CSV

## Логи

Все логи в папке `logs/`:

- `trades_YYYY-MM-DD.json` — полные данные по каждой сделке
- `trades_YYYY-MM-DD.csv` — таблица для Excel
- `signals_YYYY-MM-DD.json` — все обнаруженные сигналы
- `events_YYYY-MM-DD.log` — текстовый лог событий

## FAQ

**Q: Нужен ли API-ключ Binance?**
A: Нет. Для paper trading бот читает только публичные данные.

**Q: Бот сливает деньги?**
A: Он НЕ торгует реальными деньгами. Только симуляция. Поэтому
сначала прогони его несколько дней и посмотри результаты в логах.

**Q: Можно торговать реально?**
A: Не рекомендуется пока не убедишься в прибыльности на paper trading.
Для реальной торговли нужно будет доработать модуль исполнения ордеров.
